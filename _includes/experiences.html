My first full stack app
This web app is more of an inside joke between me and my friends, figured I'd use the joke to make my first full stack website. So the technologies I decided used to create this site was a Javascript framework called VueJs and Google's Firebase. This was my first time using VueJs as the front-end framework. VueJs allowed me to make dynamic pages, hook up a database, and authorization so that people can make and log in to their own accounts. Google's Firebase was a huge part of this project, this was also the first time really using it. Firebase was the database, authorization, and hosting for this project. After reading the docs what feels like a thousand times, finally was able to hook up everything. My first fullstack app was born!
